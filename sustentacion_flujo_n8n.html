Los dos caminos del flujo

Según el sentimiento detectado por la IA, cada cliente recibe una experiencia diferente.

😊 Cliente satisfecho
Sin cupón
La IA detectó feedback positivo o neutro. Se envía un email de seguimiento amigable para mantener la relación con el cliente.
SendCoupon = false
Va al template HTML simple
Email con mensaje personalizado
Se une con datos del cliente
Se envía por SMTP
😠 Cliente insatisfecho
Con cupón
La IA detectó insatisfacción en la reseña. Se genera un cupón de descuento automáticamente para recuperar al cliente.
SendCoupon = true
Genera código + 20% descuento
Validez: 14 días desde hoy
Template HTML con bloque cupón
Se envía por SMTP
